===============================================================================
GETTING STARTED
===============================================================================

STEP 1: Initialize Terminal Environment
---------------------------------------
1. Click the Terminal button to open a command prompt
2. Type: bash
3. Press ENTER to enter bash shell

STEP 2: Start Interactive Training Process
------------------------------------------
Type the following command and press ENTER:

   bash interactive_start_training.sh

STEP 3: Follow the Interactive Setup
------------------------------------
The script will guide you through configuration options:

Model Selection:
- Flux models
- SDXL models  
- Wan models

API Key Configuration:
- Hugging Face API key (required for Flux models)
- Gemini API key (required for video processing)

Dataset Processing:
- Image captioning options
- Video captioning options
- Training configuration review

STEP 4: Wait for Completion
---------------------------
The automated process will:
- Download required models
- Generate captions for your dataset
- Start the training process
- Save results when complete

===============================================================================
RESULTS LOCATION
===============================================================================

Your trained LoRA files will be saved in: training_outputs/

===============================================================================
PRO TIPS
===============================================================================

API Keys: Have all required API keys ready before starting the interactive 
process to avoid interruptions

First Run: Initial setup takes longer due to model downloads - be patient!

Interactive Guide: The script will walk you through each step, making it 
beginner-friendly

Model Choice: Choose your model type based on your specific use case:
- Flux: Latest high-quality diffusion models
- SDXL: Stable and well-supported option
- Wan: Specialized for certain applications

Results: Check the training_outputs/ folder for your completed LoRA files
